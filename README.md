# FCA Continuous Search

This project is a JavaScript-based program to illustrate an algorithm.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/your-username/FCA-continuous-search.git
   ```
2. Navigate to the project directory:
   ```
   cd FCA-continuous-search
   ```
3. Install the dependencies:
   ```
   npm install
   ```
4. Start the program:
   ```
   npm start
   ```

Please replace `your-username` with your actual GitHub username in the first command.
